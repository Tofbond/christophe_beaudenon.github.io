Le protocole FTP

🖥 Web Server
→ FTP →← FTP ←
💻 Your Computer

Définition / Utilisation

Le File Transfer Protocol est un protocole dédié à l'échange de fichiers sur un réseau TCP/IP.

  • Permet de copier des fichiers depuis ou vers un autre ordinateur
  • Utilisé pour administrer un site web
  • La variante sécurisée avec SSL/TLS s'appelle FTPS

Architecture FTP (Client / Serveur)

Client FTP
(ex: FileZilla)
Serveur FTP
Stockage des fichiers

Client FTP

Permet de transférer, supprimer ou modifier des fichiers sur le serveur.

Serveur FTP

Stocke et centralise les fichiers. Permet la modification à distance.

FTP Mode actif ou passif

Mode actif

Le client FTP détermine le port de connexion.

Port 21Port 20
Mode passif ✓

Le serveur détermine le port. Mieux avec les pare-feu.

Port 21High Port

  • Logiciel très utilisé pour les transferts FTP
  • Connexion avec hôte, identifiant, mot de passe et port
  • Interface : fichiers locaux à gauche / serveur à droite
  • Transfert par glisser-déposer

Avantages et inconvénients

✓ Avantages
  • Simple à utiliser
  • Très répandu dans l'administration serveur
✗ Inconvénients
  • FTP classique n'est pas sécurisé
  • Souvent remplacé par FTPS ou SFTP

Comparaison : FTP / FTPS / SFTP

FTP
Protocole historique, non sécurisé
Port 21 / 20
FTPS
FTP avec chiffrement SSL/TLS
Port 21 / 990
SFTP
Protocole basé sur SSH
Port 22

Conclusion

  • FTP est un protocole important dans l'histoire d'Internet
  • Permet de transférer des fichiers entre un client et un serveur
  • Pour la sécurité, on utilise plutôt FTPS ou SFTP

Téléphonie d'entreprise

La téléphonie d'entreprise moderne regroupe bien plus qu'un simple appel vocal :

  • Serveurs vocaux, conférences, transfert d'appels
  • Musique d'attente, visioconférence, chat en ligne
  • Gestion centralisée depuis un poste de travail

Évolution de la téléphonie

📞 Téléphone analogique

Réseau RTC traditionnel, ligne dédiée, qualité limitée.

💻 VoIP / ToIP

Voix transmise via réseau IP. Multimédia intégré : voix, vidéo, messagerie, fichiers.

Réseau collaboratif

La VoIP connecte tous les collaborateurs :

  • Communication directe entre postes sans passer par l'extérieur
  • Réseaux maillés entre sites distants
  • Intégration avec les outils métier (CRM, helpdesk...)

VoIP — Voice over IP

Définition
Transmettre la voix via des réseaux IP (Internet)
Appareils
PC, tablette, smartphone, téléphone IP
Réseau
Internet ou réseau local d'entreprise

Flux multimédias VoIP

La VoIP transporte tout type de flux multimédia :

  • 📞 Voix — appels téléphoniques
  • 🎥 Vidéo — visioconférence
  • 💬 Messagerie instantanée
  • 📁 Transfert de fichiers

IPBX — IP Private Branch eXchange

  • Permet la VoIP et donc la téléphonie d'entreprise complète
  • Reprend le rôle du standard téléphonique du PABX traditionnel
  • Peut être déployé en local ou dans le cloud
